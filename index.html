How it Works

1.

Log into Spotify

We use Spotify's secure API to connect your account with Moodify. Moodify requests permission to access only your public playlists. Read about our privacy policy below.

2.

View Your Top 10 Songs

Spotify creates a list of your 'top' songs, which we display to you. Moodify does not create this list itself, we just use Spotify's.

3.

Select Moods

For each song, use the sliders to adjust the different mood levels. Once you've done this for all songs, click Submit.

4.

Get Recommended Songs (Comming Soon)

Once we have enough testing data we will be able to recommend songs catered to your tastes and mood.

Get Started!

Privacy

We use Spotify's secure API to log into your account. The only data we request access for is your Spotify "Top" songs list, a list Spotify creates for each user. Likewise internally, we only store data on the songs themselves and the moods you assign to each song. No pesonal information or account data is kept at all.
